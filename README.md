# A-LINKS

A-LINKS is structured as a PERN stack project.

## Stack

- React Router DOM v6
- JavaScript JSX
- JWT + Express authentication
- PostgreSQL + Prisma
- Cloudinary image uploads
- npm

## Structure

- `frontend/` - Vite React frontend
- `backend/` - Express API and Prisma backend

## Development

```bash
cd frontend
npm install
npm run dev
```

```bash
cd backend
npm install
npm run dev
```
